# Website Generate Files


## Tech Stack

-   nodejs
-   typescript
-   http (express default)

## Get Started

1. clone repository

```bash
$ git clone git@github.com:ayocodingit/website-generate-files.git
```

2. install node

```bash
$ npm install
```

3. Copy `.env` from `.env.example`

```bash
$ cp .env.example .env
```
